Инструкция по установке:
	Если на компьютере установлена ОС Windows, то рекомендуется установить виртуальную машину: оперативная память 2Гб, жесткий диск 8Гб.
	На виртуальную машину установить Debian с графической оболочкой.
	
	Если на компьюьтере установлена ОС Linux (или после настройки ВМ для Windows), то:
		1. Установите GTK+ командой : "# apt-get install gtk+-3.0"
		2. Установите компилятор g++: "# apt-get install g++"
		3. Поместите файлы aux.h, window.h, window.cc, GUI.h, GUI.cc, Example.cc в каталог gui
		4. Перейдите в каталог gui командой : "cd gui"
		5. Для сборки приложения используйте команду : "g++ window.cc GUI.cc example.cc -o gui `pkg-config --cflags --libs gtk+-3.0`"
		6. Запустите исполняемый файл : "./gui"

Инструкция :
	Меню:
		Для начала игры необходимо нажать кнопку - "1".

		Программа поддерживает два режима игры: с таймером и без (тренировочный):
			Программа автоматически начинается в режиме игры без таймера,
			чтобы выбрать другой режим, нужно нажать кнопку "3".	
		Пауза:
			Чтобы поставить игру на паузу, необходимо нажать кнопку "2". 
			Чтобы снять игру с паузы, нужно нажать кнопку "1", а затем вновь запустить таймер кнопкой "3".

		Для начала новой игры нажмите "0". 
		Для выхода из программы - "b".


	Система начисления очков:
	За каждый собранный ряд начисляется 100 очков, в случае если за один ход игроку удалось собрать более одного ряда,
	начисляются дополнительные очки (5 очков за каждый дополнительный ряд).
	Таким образом одна строка - 100 очков, 2 строки - 205 и тд.

	Игра заканчивается, если :
		1) новая фигура не может быть создана
			или 
		2) достигнута высота 9 блоков по любому из столбцов.


	Управление:
		a-движение влево
		s-движение вниз
		d-движение вправо
		q-поворот против часовой
		e-поворот по часовой

	Возможные ошибки:
		Если по какой-то причине новая фигура не появляется, то нажмите "t".
